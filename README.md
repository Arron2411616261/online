# tkl

#### 环境
1.  go 1.20
2.  npm 9.5.1（版本影响不大）
3.  node.js v18.16.0（需16.0.0以上）

#### 使用说明
1.  后端启动    
在 backend 目录下  
go run main.go
2.  前端启动  
在 frontend 目录下  
npm install（仅需执行一次）  
npm run dev

#### 安装教程
1.  go导入gin：  
go get -u github.com/gin-gonic/gin

下面为数据库相关包  

2.  go导入gorm：  
go get -u http://github.com/jinzhu/gorm
3.  go导入gorm.io：  
go get -u gorm.io/gorm  
go get -u gorm.io/driver/mysql
