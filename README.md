#### 环境

1.  go 1.20
3.  node.js v18.16.0（需16.0.0以上）

#### 安装依赖
1. 在frontend目录下  
   npm install element-plus unplugin-vue-components unplugin-auto-import sass axios @vueuse/core pinia-plugin-persistedstate --save


#### 启动项目
1.  后端启动    
在 backend 目录下  
go run main.go
2.  前端启动  
在 frontend 目录下   
npm run dev  

#### 技术栈

1. 后端  
   go + gin + gorm + mysql
2. 前端
   vue3 + vite +  pinia + vue router  + element plus + axios
